La Pistola

Una pistola es un arma de fuego corta diseñada para ser apuntada y disparada con una sola mano, o con dos,se puede utilizar
para la caza dependiendo de él arma

Sin embargo, a pesar de que una pistola está diseñada para ser disparada con una sola mano, no es obligatorio que se utilice
de esa manera.
